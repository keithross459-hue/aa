# Launch Checklist

- Production env vars set and reviewed.
- Stripe live checkout, portal, invoice PDF, and webhook tested.
- Admin owner account created and protected.
- Pro and CEO plan gates verified.
- Free plan generation gate verified.
- Referral signup, commission, payout request, admin approval, and paid state verified.
- One-click machine creates product, campaign, listings, referral activation, PDF, and ZIP.
- Executive dashboard loads for admins.
- Cookie consent appears once and persists.
- Account export downloads JSON.
- Mobile navigation works.
- Legal endpoints reviewed by counsel before public launch.
- Backups scheduled and restore tested.
- Uptime monitor and Sentry alerts live.
- Rollback plan rehearsed.
